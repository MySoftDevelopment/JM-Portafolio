# JuanMolina Portafolio 2026.

Estás visualizando el repositorio de la parte del frontend de JuanMolina Portafolio.

Este frontend está programado usando React + TypeScript + Vite.

## ¿Cómo usar?

- Principalmente necesitas descargar [Node](https://nodejs.org/es/download/).
- Instalado Node ya podrás usar npm para ejecutar el siguiente comando:
```
npm install --global yarn
```
- Con yarn instalado podrás ejecutar yarn para descargar las dependencias del proyecto.
- Para poder ejecutar el proyecto podrás usar los siguientes comandos:
```
yarn start
```
```
yarn build
```
